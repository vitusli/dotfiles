# About Render Raw

Render Raw is a Blender add-on for perfecting the colors in your render. 

You can download Render Raw by [purchasing it on the Blender Market](https://blendermarket.com/creator/products/render-raw). 

Watch **[this tutorial](https://www.youtube.com/watch?v=hS7uaTquwWc)** for an overview of why the raw workflow is important. 

The minimum supported Blender version is 4.1, though it will work with 4.0 albeit with a few bugs since the viewport compositor recieved several stability fixes in Blender 4.1. 

For more about how Render Raw works and how to use it, continue on to the Enabling Render Raw page.
